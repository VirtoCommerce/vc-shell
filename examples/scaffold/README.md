# VC-Shell Scaffold Example

This directory is the canonical source for generator-derived documentation snippets.

The contents mirror the starter shape produced by `create-vc-app` and `/vc-app generate` for a minimal module. Documentation should include snippets from this directory when it needs reproducible starter code instead of a product-specific example.

When scaffold templates change, update this example in the same pull request and run type checking for the example app.
